# VGR Multipage Website

## Before uploading
1. Copy your existing `vgr-images-white.png`, `tyler-gonzalez.jpeg`, `william-tally.jpeg`, `steven-clemons.jpeg`, and `caleb-bacon.jpeg` into the `IMAGES` folder.
2. In `js/config.js`, replace `YOUR_ELFSIGHT_APP_ID` with the ID from your Elfsight Instagram Feed widget.
3. Before each race, replace `youtubeVideoId` with the official IMSA YouTube video ID.
4. Upload every file and folder to the root of your GitHub repository.

## Pages
Home, Team, Drivers, Cars, Schedule, News, Media, Live, Sponsors, Contact.

## Language
English is the default. Use the USA and Puerto Rico flag buttons to change language.
